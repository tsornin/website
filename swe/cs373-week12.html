This week in class we covered reflection and singletons. No more projects! The only major things left for this class are the project presentation and the third exam.

This weekend I installed Stephan T. Lavavej's MinGW Distro. The newest version includes, among other things, GCC 4.6. The feature with the most immediate benefits (for me, at least) is the "range-based for loop", which has long been missing from C++. Instead of this ugly thing...

int sum ( const std::vector < int >& xs ) {
     int result = 0;
     for ( std::vector < int >::const_iterator
     i = xs.begin(); i != xs.end(); ++i ) {
          int x = *i;
          result += x;
     }
     return result;
}

...we now have a much more Java- or Python-like syntax.

int sum ( const std::vector < int >& xs ) {
     int result = 0;
     for ( const int& x : xs ) {
          result += x;
     }
     return result;
}

Awesome.
